
# 🛒 MERN E-Commerce Application





A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express, React, Node.js).
This project allows users to browse products, add items to cart, place orders, and manage their accounts, while admins can manage products, orders, and homepage banners.
## Live Demo 

⚙️ Backend: https://e-commerce-backend-lgb3.onrender.com
## **📌 Features:**

  👤 User Features

- ⚡ User Registration & Login (JWT Authentication).
- 🎯 Browse Products with Filters & Search.
- 🧩 Add to Cart & Manage Cart.
- 🌐 Add/Edit/Delete Address.
- 📱Add Product Reviews.

## **🛠️ Admin Features:**

- Admin Dashboard.
- Add / Edit / Delete Products.
- Upload Product Images (Cloudinary).
- Manage Orders (Update Status).
##   ⚙️ Setup Instructions
1. Clone the Repository-
git clone- https://github.com/Shubham-Kumar-Rai/E-commerce-backend.git

2. cd backend
- npm install
- Create .env file in backend:
- PORT=5000
- MONGO_URI=your_mongodb_connection_string
- JWT_SECRET=your_secret_key
- CLOUDINARY_CLOUD_NAME=your_cloud_name
- CLOUDINARY_API_KEY=your_api_key
- CLOUDINARY_API_SECRET=your_api_secret
- Run backend: nodemon server.js or node server.js
